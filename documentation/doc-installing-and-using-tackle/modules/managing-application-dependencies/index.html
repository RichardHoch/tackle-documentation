Managing application dependencies

You can add, delete, and view application dependencies in the Application Inventory page of the Tackle web console.

Prerequisites

  • The Application Inventory must contain at least two applications.

Procedure

  1. Click Application Inventory.

  2. Click the Options menu beside an application and select Manage dependencies.

  3. Select applications in the northbound or southbound dependencies fields.

    Note

    To remove dependencies, delete the applications in the dependencies fields.

  4. Click Close.
