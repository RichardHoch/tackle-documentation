Tagging applications

You can tag one or more applications in the Application Inventory page of the Tackle web console.

Prerequisites

  • The Application Inventory must contain at least one application.

  • The Controls must contain at least one tag.

Procedure

  1. Click Application Inventory.

  2. Select one or more applications.

  3. Click the Options menu in the toolbar and select Add tag.

  4. Select one or more tags and then click Apply tags.

  5. Expand an application to view its details.

    The display order of the tags is determined by the Rank of the tag type.
