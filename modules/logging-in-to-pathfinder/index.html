Logging in to Tackle

You can log in to the Tackle web console.

Prerequisites

  • You must have Tackle installed.

Procedure

  1. Log in to the OpenShift Container Platform web console.

  2. Click Networking → Routes.

  3. Select the {project-name} project in the Project: list.

  4. Click the URL for the tackle-ui service to open the login page for the Tackle web console.

  5. Enter your Username and Password and click Log in.

    The Tackle web console is displayed.
